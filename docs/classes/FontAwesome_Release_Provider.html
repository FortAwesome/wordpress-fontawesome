get_resource_collection()

get_resource_collection(string  $version, mixed  $style_opt, array  $flags = array('use_pro' => false, 'use_svg' => false, 'use_shim' => true)) : array

Gets an array containing version, shim, source URLs and integrity keys for given params.

They should be loaded in the order they appear in this collection.

Parameters

string $version
mixed $style_opt

either the string 'all' or an array containing any of the following: ['solid', 'regular', 'light', 'brands']
array $flags

boolean flags, defaults: array('use_pro' => false, 'use_svg' => false, 'use_shim' => true)

Throws

\InvalidArgumentException

if called with use_svg = true, use_shim = true and version < 5.1.0. shims were not introduced for webfonts until 5.1.0. Throws when called with an array for $style_opt that contains no known style specifiers.

Returns

array

previous_minor_release()

previous_minor_release() : string|null

Returns a version number corresponding to the minor release immediately prior to the most recent minor release.

Returns

string|null —

latest patch level for the previous minor version. major.minor.patch version (not a semver). Returns null if there is no latest (and therefore no previous). Returns null if there's no previous, because the latest represents the only minor version in the set of available versions.
